Applicazione web per WeStudents

Introduzione

Come progetto per il Master in Programmazione Web ho deciso di realizzare un'applicazione web, che faccia da esempio sul come implementare le già esistenti app mobile di WeStudents sul web.

Ho scelto di realizzare un progetto per la startup WeStudents perchè il tipo di prodotto che offrono è stato uno strumento fondamentale per gestire la mia vita scolastica durante le scuole superiori.

Ho optato per un'applicazione full-stack in modo da comprendere tutte le conoscenze acquisite durante il mio percorso di apprendimento, intrapreso prima e durante le Super Guide di start2impact.

Come installare

Tecnologie

  • Node.js 10+
  • NPM o altro package manager per Node.js
  • MySQL 5.7

Procedura

  1. Creare un database e un utente per gestirlo e aggiornare i dati nel file ./server/config/mysql.js. 
              CREATE DATABASE westudents;
          CREATE USER 'westudents_user'@'localhost' IDENTIFIED BY 'password_sicura';
          ALTER USER 'westudents_user'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password_sicura';
          GRANT ALL PRIVILEGES ON westudents.* TO 'westudents_user'@'localhost';
          FLUSH PRIVILEGES;
  2. Eseguire le query contenute nel file create-table-mysql.sql per creare le tabelle.
  3. Stare in questa cartella (web-student-app/).
  4. Intallare le dipendenze per il server con il comando npm i.
  5. Eseguire il server con il comando npm start.
  6. In un altro terminale accedere al client con cd client.
  7. Intallare le dipendenze per il client con il comando npm i.
  8. Eseguire il client con il comando npm run serve e accedere alla pagina http://localhost:8080.

Tecnologie utilizzate

Lato client

  • HTML: struttura delle pagine e template di Vue.js.
  • CSS: stile delle pagine.
  • Vue.js (JavaScript): costruzione dell'applicazione web lato client, dinamicità delle pagine e gestione delle chiamate all'API lato server con axios.
  • WebStorage API: utilizzata per salvare i dati della sessione sul browser.

Lato server

  • Node.js (JavaScript): costruzione dell'applicazione web lato server.
  • Express: creazione dellaRESTful API.
  • JWT: gestione dei token per le sessioni e la verifica dell'ownership delle risorse.
  • Sequelize: gestione delle chiamate al database.
  • MySQL: DBMS per database relazionali per salvare in modo permanente i dati dell'applicazione.
